## 개발목표
- mvc -> mvc2 -> spring 게시판 변환
- 게시판 주요기능

## 개발환경
- Eclipse 2019-12(4.14.0)
- Tomcat 8.5
- JDK 1.8.0
- Oracle11

## 프레임워크
- SPRING 4.3.29

## 라이브러리
- Tiles
- Mybatis
- Spring Security

## 구현기능
- 게시판 CRUD
- Ajax를 활용한 대댓글
- 게시판 페이지네이션
- 파일 업로드 & 다운로드
- AOP를 활용한 로그
- Security를 활용한 필터 및 인증
